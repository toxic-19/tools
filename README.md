## 🚀 已包含的工具

### 1. JSON Toolkit Pro (Chrome 扩展)
一个功能强大的 JSON 处理工具，支持格式化、修复、对比以及折叠视图。

#### 核心功能：
- **格式化/压缩**：一键美化或压缩 JSON 字符串。
- **自动修复**：修复非标准 JSON（如单引号、尾部逗号等）。
- **JSON 反转义**：处理包裹在字符串中的 JSON。
- **对比模式**：可视化的 JSON 差异对比。
- **树状视图**：支持对象和数组的层级折叠。
- **多主题**：支持深色/浅色模式切换。
- **全屏模式**：支持在大屏幕下处理复杂数据。

#### 使用方法：
进入 `json-extension` 目录查看详情，或直接在 Chrome 中加载该文件夹。
