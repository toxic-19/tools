# JSON Toolkit Pro — 开发者必备的 JSON 工具箱

一个现代视图、功能全面的浏览器扩展，旨在为开发者提供最高效的 JSON 处理体验。

## 🏆 核心功能

### 1. 格式化与美化 (Formatter)
- **实时校验**：输入时自动检查合法性，显示错误位置及详细信息。
- **一键格式化**：自动缩进（2 空格），让代码清晰易读。
- **一键压缩 (Compact)**：去除空格换行，生成最小体积的字符串。
- **树状折叠视图 (Tree View)**：深度支持对象与数组的展开/收起，大文件阅读利器。

### 2. 增强修复与处理
- **自动修复 (Repair)**：支持修复单引号、缺少引号、尾部多余逗号、注释、未引用的键名等常见非标准 JSON。
- **JSON 反转义 (Unescape)**：将转义后的字符串（如 `\"` 包裹的文本）一键还原为标准对象。

### 3. 数据对比 (JSON Diff)
- **行级对比**：采用 LCS 算法识别新增、删除、修改的行。
- **高亮显示**：绿色标记新增，红色标记删除，黄色标记修改，并实时统计差异数。
- **同步滚动**：左右分屏联动，对比体验丝滑。

### 4. 高级交互
- **全屏模式**：在新标签页中打开，利用更大空间处理超大规模 JSON。
- **主题切换**：支持深色/浅色模式一键切换。
- **快捷操作**：支持 Tab 键缩进、双击括号折叠、一键复制到剪贴板。

## 🚀 安装方法

1.  下载或克隆本项目：`git clone https://github.com/toxic-19/tools.git`
2.  进入浏览器（Chrome/Edge）的 **扩展程序管理** 页面 (`chrome://extensions/`)。
3.  点击右上角开启 **“开发者模式”**。
4.  点击 **“加载已解压的扩展程序”**。
5.  选择项目中的 `json-extension` 文件夹即可完成安装。
